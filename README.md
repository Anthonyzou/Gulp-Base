### A base repository for a gulp based build system for web

Run with

    > npm install
    > gulp

File types go in these directories.

    ASSETS:       src/assets/**/*
    COFFEE:       src/js/main/**/*.coffee
    EXTERNAL_JS:  src/js/external
    HTML:         src/**/*.html
    JADE:         src/**/*.jade, !src/**/_*.jade
    LESS:         src/styling/**/*.less
    TARGET:       build
